Html문서


HTML
▶ * 웹에서 볼 수 있는 문서를 만들기 위한 일종의 표준 언어이다.
* 일반적인 텍스트로 되어 있으며 태그(Tag)를 통해 웹 브라우저로 보이는 모양을 나타낸다.
* 파일 확장자는 .html 또는 .htm이다.

HTML 문서의 기본적인 형식
① 시작 태그와 끝 태그 : 시작 태그(형식 : 〈태그 이름〉)와 끝 태그(형식 :〈/태그 이름〉)로 이루어진다. (예) 〈HTML〉…〈/HTML〉
② 문서의 내용 : 시작 태그와 끝 태그 사이에 문서의 내용이 온다.

③ 문서의 속성 : 시작 태그 안에 속성 이름＝"속성값" 의 형태를 사용하여 해당 문서 내용의 속성을 나타낼 수 있다. (예) 〈FONT COLOR＝"RED"〉 : 글자의 색을 빨강색으로 지정

글자 및 양식 관련 태그
① 문서의 헤더 : 문서에서 제목에 해당하는 문자열을 표시하는 태그로, 〈Hn〉…〈/Hn〉(n＝1, 2, 3, 4, 5, 6) 으로 표시하며, 숫자가 클수록 크기는 작아진다.
② 문자의 크기 : 〈FONT SIZE＝n〉…〈/FONT〉(n＝1, 2, 3, 4, 5, 6, 7)
③ 줄바꾸기 : 〈BR〉(워드 프로세서에서 키를 누르는 것과 같은 기능)
④ 문단나누기 : 〈P〉…〈/P〉 또는 〈P〉(줄을 바꾸면서 그 다음 한 줄을 공백 처리)


▶ FONT 태그 : FONT는 글자의 크기, 색깔, 서체를 지정하는 태그이다.
* 크기 : 〈FONT SIZE = "n"〉
* 색깔 : 〈FONT COLOR = "#색상 코드 또는 색상명"〉
* 서체 : 〈FONT FACE = "글꼴체"〉


태그 입력시 주의 사항
② 수식 키를 인식하지 못하므로 문단을 바꾸거나 줄을 바꾸려면 해당 태그를 입력해야 한다.
③ 두 개 이상의 연속된 빈 칸을 인식하지 못하므로 두 번째 빈 칸부터 해당 태그를 입력해야 한다.
④ 브라우저의 종류에 관계 없이 공통적으로 실행 가능한 태그를 사용하는 것이 좋다.


목록 및 표 작성 ① 목록 작성 : 목록은 작업의 순서나 항목 등을 차례로 나열할 때 사용한다.
　　㉠ 번호가 없이 나열되는 목록에는 〈UL〉…〈/UL〉을, 순서가 있는 목록에는 〈OL〉…〈/OL〉을 사용한다.
　　㉡ 나열되는 각 항목은 〈LI〉 태그를 이용하여 표기한다.


② 표 작성 : 〈TABLE〉 태그를 사용한다.
　　㉠ 〈TR〉…〈/TR〉 : 표의 행을 한 줄 추가한다.
　　㉡ 〈TD〉…〈/TD〉 : 표 안에 들어갈 데이터를 표시한다(표의 열에 해당함.).
　　㉢ 〈TH〉…〈/TH〉 : 표 안에 들어갈 열의 제목을 표시한다.
　　㉣ 〈TABLE BORDER〉 : 표 테두리의 굵기를 지정한다.
　　㉤ 〈CAPTION〉 : 표의 제목을 표시해 준다.


표 작성하기
 * ALIGN : 표 안의 글자 배열 * ROWSPAN : 행을 합치는 속성 * COLSPAN : 열을 합치는 속성

③ HTML 문서의 연결 : 〈A〉 태그
　　㉠ 문서 간의 연결 : 〈A HREF＝"파일 이름" 또는 "URL"〉…〈/A〉를 사용하여, 하이퍼링크를 클릭했을 때 연결되는 문서를 지정할 수 있다.
　　㉡ 문서 내의 연결 : 〈A HREF＝"#이름"〉…〈/A〉과 〈A NAME＝"이름"〉…〈/A〉를 사용하여 문서 내의 특정한 위치를 지정하여 이동할 수 있다.
　　㉢ 편지쓰기 : 〈A HREF = "MAILTO : 수신자 메일 주소"〉…〈/A〉


프레임
웹 브라우저의 화면을 여러 개로 나눈 단위
① 〈FRAME SRC＝"URL" 또는 "HTML 파일명" NAME＝"프레임 이름"〉 : 분할된 각 프레임의 이름을 정해 놓고 지정된 문서가 NAME으로 지정한 위치에 나타나게 해 준다.
② 〈FRAMESET COLS＝"픽셀수 또는 백분율"〉 : 화면을 좌우로 분할
③ 〈FRAME SRC＝"URL" 또는 "HTML 파일명" NAME＝"프레임 이름"〉 : 분할된 각 프레임의 이름을 정해 놓고 지정된 문서가 NAME으로 지정한 위치에 나타나게 해 준다.

메모장을 이용한 HTML 문서 작성
① 메모장에서 HTML 태그를 작성한다.
② ［파일］-［다른 이름으로 저장］을 선택하여 파일 형식을 '모든 파일'로 선택하고, 파일 이름을 '*.html' 로 한다.
③ 탐색기에서 파일을 실행하거나, 웹 브라우저를 실행시킨 후 파일을 불러 와 확인한다.


이미지 삽입하기 〈IMG SRC＝"이미지 파일 이름"〉으로 이미지를 삽입할 수 있으며, JPG, GIF, PNG 등의 이미지만 사용할 수 있다.

02.HTML 태그를 활용하여 표를 작성하는 방법을 설명한 것으로 옳은 것을 모두 고르면?

1. 표를 만들 때에는 〈TABLE〉태그를 사용한다.
2. 〈TH〉 … 〈/TH〉는 표 안에 들어갈 데이터를 표시한다.
3. 〈TR〉 … 〈/TR〉는 표의 행을 한 줄 추가할 때 사용한다.
4. 〈CAPTION〉 태그는 표의 제목을 표시해 주는 역할을 한다.
5. 〈TABLE BORDER〉는 표 안 글자의 굵기를 지정할 때 사용하는 태그이다.
